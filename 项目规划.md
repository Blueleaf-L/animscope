# 中国动画公司作品分析平台 — 项目总结

## 最终方案

**纯静态网站 + GitHub Pages 部署**，无后端、无数据库、零运维成本。

V2 最初规划为前后端分离（FastAPI + PostgreSQL），经过实际部署验证后发现：
- 免费云服务大多需要信用卡实名，国内用户门槛高
- 数据量小（72公司/586作品），完全可以在客户端完成所有计算
- GitHub Pages 和 V1 一样稳定可靠

最终选择回归静态方案：Python 脚本一次性预计算所有分析结果 → 输出 JSON + PNG → 前端直接加载。

## 项目结构

```
├── index.html / css / js / pages/    # 前端源码
├── data/static/                       # 预计算静态数据（JSON + PNG）
├── scripts/
│   ├── convert_excel_to_json.py       # Excel → 结构化 JSON
│   └── build_static.py               # JSON → 分析结果 + 图表
├── data/
│   ├── companies.json / works.json    # 清洗后的源数据
│   └── *.xlsx                        # 原始 Excel（不入库）
└── docs/                              # GitHub Pages 部署目录
```

## 数据流

```
Excel (.xlsx)
  → [convert_excel_to_json.py]  清洗合并单元格、评级映射
  → companies.json + works.json
  → [build_static.py]           numpy/scipy 统计 + matplotlib 图表
  → data/static/*.json + charts/*.png
  → 前端 fetch 加载 → ECharts 渲染
```

## 技术选型

| 层 | 技术 | 用途 |
|---|---|---|
| 数据清洗 | pandas + numpy | Excel 读取、合并单元格处理、评级映射 |
| 统计分析 | numpy + scipy | Z-Score、Cohen's d、线性回归、分位数 |
| 图表预渲染 | matplotlib + seaborn | 评级分布图、箱线图、热力图 PNG |
| 交互式图表 | ECharts 5 CDN | 环形图、气泡图、雷达图、时间线 |
| 前端 | Vanilla HTML/CSS/JS | SPA、Hash 路由、暗色模式、响应式 |
| 部署 | GitHub Pages | 免费、零运维、全球 CDN |

## 页面功能

| 页面 | Hash | 数据源 | 图表 |
|------|------|--------|------|
| 首页概览 | `#overview` | overview.json | ECharts 环形图 + 玫瑰图 + 双轴趋势 |
| 公司总览 | `#companies` | companies_lite.json | ECharts 气泡图 + 表格 |
| 公司详情 | `#company/:id` | companies_full.json | ECharts 雷达图 + 散点时间线 |
| 作品排行 | `#rankings` | rankings_*.json | 卡片网格 |
| 趋势分析 | `#trends` | trends.json | ECharts 折线 + 热力图 |
| 公司对比 | `#compare` | 客户端计算 | ECharts 多雷达 + 柱状图 + 差异面板 |
| 作品搜索 | `#search` | 客户端搜索 | 表格 + 分页 |
| 分析报告 | `#report` | 预渲染 PNG | matplotlib 图表 |
