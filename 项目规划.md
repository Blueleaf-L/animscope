# 中国主流网络动画公司作品分析平台 — 项目总结

## 最终方案

纯静态网站 + GitHub Pages 部署，无后端、无数据库、零运维成本。

## 项目结构

```
├── index.html / css / js/        # 网站文件
├── data/static/                   # 预计算静态数据（JSON + PNG）
├── scripts/
│   ├── convert_excel_to_json.py   # Excel -> JSON（含类型人工校对）
│   └── build_static.py           # JSON -> 分析结果 + 图表 PNG
├── frontend/                      # 前端源文件
├── data/
│   ├── companies.json / works.json
│   └── 公司的完整作品及对应制作类型.json
├── README.md
└── 项目规划.md
```

## 数据流

```
Excel (.xlsx)
  -> convert_excel_to_json.py  (清洗、评级映射、类型校对)
  -> companies.json + works.json
  -> build_static.py           (numpy/scipy 统计 + matplotlib 图表)
  -> data/static/*.json + charts/*.png
  -> 前端 fetch 加载 -> ECharts 渲染
```

## 技术选型

| 层 | 技术 | 用途 |
|---|---|---|
| 数据清洗 | pandas + numpy | Excel 读取、合并单元格处理、评级映射 |
| 统计分析 | numpy + scipy | Z-Score、Cohen's d、线性回归、分位数 |
| 图表预渲染 | matplotlib + seaborn | 评级分布图、箱线图、热力图 PNG |
| 交互式图表 | ECharts 5 CDN | 环形图、气泡图、雷达图、时间线、热力图 |
| 前端 | Vanilla HTML/CSS/JS | SPA、Hash 路由、暗色模式、响应式 |
| 部署 | GitHub Pages | 免费、零运维 |

## 页面功能

| 页面 | Hash | 数据源 | 图表 |
|------|------|--------|------|
| 首页概览 | `#overview` | overview.json | 环形图 + 玫瑰图 + 双轴趋势 |
| 公司总览 | `#companies` | companies_lite.json | 气泡图 + 表格 |
| 公司详情 | `#company/:id` | companies_full.json | 雷达图(中英双语) + 散点时间线 |
| 公司排行 | `#rankings` | rankings_*.json | 卡片网格 + Z-Score |
| 趋势分析 | `#trends` | trends.json | 折线 + 热力图(点击下钻) |
| 公司对比 | `#compare` | 客户端计算 | 多雷达 + 柱状图 + Cohen's d |
| 作品搜索 | `#search` | 客户端搜索 | 表格 + 智能分页 |
| 分析报告 | `#report` | 预渲染 PNG | matplotlib 图表 |
